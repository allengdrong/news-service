# 📰 News Service

> 여러 언론사의 최신 뉴스를 자동으로 수집하고,  
> 빠르게 검색할 수 있는 뉴스 검색 서비스

🔗 **서비스 바로가기**  
https://news-service-vatr.onrender.com/

---

## 어떤 서비스인가요?

**News Service**는 여러 언론사의 RSS 뉴스를 자동으로 수집해서  
한 화면에서 최신 기사들을 보여주고, 검색까지 가능한 뉴스 서비스입니다.

매번 포털 사이트를 돌아다니지 않아도,  
실시간으로 업데이트되는 뉴스를 한 곳에서 확인할 수 있도록 만들었습니다.

---

## 주요 기능

### 1. 최신 뉴스 자동 수집
- 언론사의 RSS를 주기적으로 수집합니다.
- 새로운 기사는 자동으로 업데이트됩니다.

---

### 2. 빠른 검색 기능
- 제목 + 요약 기준 검색
- 키워드 기반 기사 조회
- 최신순 정렬

---

### 3. 고성능 검색 엔진
Elasticsearch 기반 검색으로:

- 데이터가 많아져도 빠른 응답 속도
- 전문 검색 최적화
- 실무에서 사용하는 검색 구조 적용

---

## 이런 분들께 좋아요

- 여러 언론사 뉴스를 한 번에 보고 싶은 분
- 키워드로 뉴스 검색이 필요한 분
- 개인 프로젝트 / 포트폴리오 참고용 서비스 찾는 개발자

---

## 기술 스택

### Backend
- Java 17
- Spring Boot 3
- Gradle

### Search
- ElasticSearch (Elastic Cloud)

### Infra
- Docker
- Render (Free Tier)

---

## 시스템 구조

RSS (언론사) > 뉴스 수집 스케줄러 > Elasticsearch 색인 > Spring Boot API > 웹 화면 제공

Elasticsearch가 초기화되거나 장애가 발생해도  
RSS를 다시 수집해서 자동으로 복구되는 구조입니다.
